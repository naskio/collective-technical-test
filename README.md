# collective-technical-test
Collective - Technical test using NestJS and ReactJS

## Configuration

- cp ```.env_example``` to ```.env```
    ```shell
    cp .env_example .env
    ```

## Build

```shell
docker-compose build
```

## Run

```shell
docker-compose up -d 
```

in order to stop the container:

```shell
docker-compose down
```
